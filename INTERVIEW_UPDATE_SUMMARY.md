# Сводка обновления: Функции для собеседований

## 🎯 Что добавлено

Voice Assistant Overlay теперь специально оптимизирован для помощи на технических собеседованиях!

### Основные функции

1. **4 режима работы:**
   - 🎯 Собеседование - краткие технические ответы с контекстом
   - 🧮 Алгоритмы - помощь с алгоритмическими задачами
   - 📝 Шпаргалка - быстрые факты и определения
   - 💬 Общий - стандартный режим

2. **Контекстная память:**
   - Сохранение последних 10 вопросов в режиме собеседования
   - Более релевантные ответы с учетом контекста
   - Автоматическая очистка при переключении режима

3. **Адаптивные промпты:**
   - Оптимизированные системные промпты для каждого режима
   - Краткие ответы (2-3 предложения)
   - Фокус на ключевых концепциях

## 🚀 Как использовать

### Быстрый старт

1. Запустите приложение
2. Выберите режим через выпадающее меню (по умолчанию: 🎯 Собеседование)
3. Задавайте вопросы через микрофон
4. Получайте краткие ответы в overlay окне

### Переключение режимов

```
Теоретические вопросы → 🎯 Собеседование
Coding challenge       → 🧮 Алгоритмы
Быстрые факты         → 📝 Шпаргалка
Общие вопросы         → 💬 Общий
```

### Управление контекстом

- **Просмотр истории:** Клик на режим → видно количество вопросов
- **Очистка истории:** Клик на режим → "🗑️ Очистить историю (N)"
- **Автоочистка:** При переключении режима

## 📊 Примеры

### Режим собеседования
```
Вопрос: "Что такое замыкание в JavaScript?"
Ответ: "Замыкание - это функция, которая имеет доступ к переменным 
из внешней области видимости, даже после завершения выполнения 
внешней функции. Используется для инкапсуляции данных."
```

### Режим алгоритмов
```
Вопрос: "Найти два числа в массиве с суммой target"
Ответ: "Используй HashMap для хранения чисел. Для каждого элемента 
проверяй, есть ли (target - element) в HashMap. Сложность: O(n) время, 
O(n) память.

Псевдокод:
map = {}
for num in array:
  if (target - num) in map: return [num, target-num]
  map[num] = true"
```

### Режим шпаргалки
```
Вопрос: "HTTP методы"
Ответ: "GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS, CONNECT, TRACE"
```

## 🔧 Технические детали

### API изменения

**POST /api/generate** теперь принимает:
```json
{
  "transcript": "Что такое closure?",
  "mode": "interview",
  "context": [
    {"question": "...", "answer": "..."}
  ]
}
```

### Новые типы

```typescript
type AssistantMode = 'general' | 'interview' | 'algorithm' | 'cheatsheet';

interface InterviewContext {
  questions: Array<{
    question: string;
    answer: string;
    timestamp: number;
  }>;
  startTime: number;
  topic?: string;
}
```

### Store действия

```typescript
setMode(mode: AssistantMode)
addToInterviewContext(question: string, answer: string)
clearInterviewContext()
getInterviewContext(): Array<{question: string, answer: string}>
```

## 💡 Советы

### Для эффективного использования

✅ **Делайте:**
- Формулируйте вопросы четко и конкретно
- Используйте правильный режим для типа вопроса
- Очищайте историю при смене темы

❌ **Избегайте:**
- Слишком широких вопросов
- Многосоставных вопросов
- Неконкретных формулировок

## 🐛 Известные ограничения

- Контекст ограничен 10 вопросами
- История не сохраняется между сеансами
- Оптимизировано только для русского языка
- Нет горячих клавиш для переключения режимов
