# Сводка улучшений системы захвата и обработки аудио

## 1. Pre-roll буфер (захват начала речи)

**Проблема:** Первое слово интервьюера не попадало в запись.

**Решение:** Добавлен кольцевой буфер на 3 секунды, который постоянно хранит последние аудио данные.

**Результат:** Начало речи всегда захватывается полностью.

---

## 2. Накопление речи и детекция пауз

**Проблема:** Один вопрос разбивался на несколько частей.

**Решение:** Система накапливает аудио во время речи и обрабатывает только после 2 секунд тишины.

**Результат:** Вопросы обрабатываются целиком, лучшее качество транскрипции.

---

## 3. Обновление моделей до GPT-5

**Изменение:** Использование последних моделей OpenAI.

- **gpt-5** - для генерации ответов
- **gpt-5-mini** - для детекции вопросов

**Результат:** Более качественные ответы и лучшее понимание контекста.

## Архитектура системы

```
┌─────────────────────────────────────────────────────────────┐
│  WASAPI Background Thread (всегда работает)                 │
│  ├─► Pre-roll Buffer (3 сек) - кольцевой буфер             │
│  └─► Main Buffer (до 30 сек) - при активной записи         │
└─────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│  Мониторинг каждые 5 секунд                                 │
│  ├─► get_audio_level() - проверка без очистки               │
│  ├─► Голос? → Накапливаем (состояние: "Говорите")          │
│  ├─► Тишина? → Ждем 2 сек (состояние: "Пауза")             │
│  ├─► 2 сек тишины → Ждем 500ms (post-roll)                 │
│  └─► Post-roll → Обработка (состояние: "Слушаю")           │
└─────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│  Обработка накопленного аудио                               │
│  ├─► Whisper API - транскрипция                             │
│  ├─► GPT-5-mini - детекция вопроса                          │
│  └─► GPT-5 - генерация ответа                               │
└─────────────────────────────────────────────────────────────┘
```

---

## Параметры системы

| Компонент | Параметр | Значение | Настраивается |
|-----------|----------|----------|---------------|
| Pre-roll | Размер буфера | 3 секунды | Нет |
| Post-roll | Задержка | 500ms | Нет |
| Main buffer | Максимум | 30 секунд | Нет |
| VAD | Порог голоса | 0.02 (2%) | Да (UI) |
| Паузы | Длительность | 2 секунды | Нет |
| Мониторинг | Интервал | 5 секунд | Нет |

- 🎤 **Говорите...** (зеленый) - активная речь, накопление
- ⏸️ **Пауза** (оранжевый) - тишина, ожидание завершения
- 👂 **Слушаю...** (серый) - ожидание начала речи

- 30 полос с центральным акцентом
- Реальное время (обновление каждые 50ms)

## Преимущества

✅ **Полный захват речи** - начало (pre-roll 3 сек) и конец (post-roll 500ms) не теряются  
✅ **Целостность вопросов** - один вопрос = одна обработка  
✅ **Лучшая транскрипция** - больше контекста для Whisper  
✅ **Меньше ложных срабатываний** - фильтрация шума  
✅ **Качественные ответы** - GPT-5 с контекстом  
✅ **Визуальная обратная связь** - пользователь видит состояние  
✅ **Надежность** - последнее слово всегда захватывается
