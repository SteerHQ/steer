## 🎨 Дизайн референсы

### LockedIn AI особенности:
1. **Минималистичный overlay** - небольшое окно с ответом
2. **Быстрый доступ** - hotkey для показа/скрытия
3. **Контекстные подсказки** - понимает контекст разговора
4. **Краткие ответы** - bullet points, не эссе
5. **Профессиональный вид** - не отвлекает от собеседования

## 🚀 Быстрый старт для разработки

### 1. Создать новую ветку
```bash
git checkout -b feature/lockedin-ui
```

### 2. Установить дополнительные зависимости
```bash
cd apps/desktop
bun add framer-motion react-markdown remark-gfm
bun add -D @types/react-markdown
```

### 3. Начать с улучшения overlay
- Создать новый компонент `ModernOverlay`
- Добавить glassmorphism стили
- Реализовать drag & drop
- Добавить анимации

### 4. Улучшить промпты
- Создать `prompts/` директорию
- Добавить специализированные промпты
- Реализовать определение типа вопроса

## 🎯 Конкурентные преимущества

Что сделает ваш продукт лучше LockedIn AI:

1. **Открытый исходный код** - сообщество может улучшать
2. **Локальная обработка** - приватность данных
3. **Кастомизация** - пользователи могут настраивать под себя
4. **Бесплатная версия** - с ограничениями, но функциональная
5. **Русский язык** - поддержка с первого дня
6. **Расширяемость** - плагины и интеграции
