# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено

- **🎯 Режимы работы для собеседований:**
  - Режим "Собеседование" - краткие технические ответы с контекстом
  - Режим "Алгоритмы" - помощь с алгоритмическими задачами
  - Режим "Шпаргалка" - быстрые факты и определения
  - Режим "Общий" - стандартный режим
- **Контекстная память** - сохранение последних 10 вопросов в режиме собеседования
- **Компонент InterviewMode** - UI для переключения режимов
- **InterviewService** - сервис для работы с режимами и контекстом
- **Адаптивные промпты** - оптимизированные системные промпты для каждого режима
- **Документация:**
  - `docs/interview-modes.md` - подробное описание режимов
  - `docs/examples-ru.md` - примеры использования на собеседованиях
- Обновленный `usage-guide-ru.md` с информацией о режимах
- CI/CD пайплайн с GitHub Actions для автоматической проверки кода
- Prettier для форматирования кода
- ESLint конфигурация для TypeScript и React
- Vitest для unit-тестирования
- Централизованная система обработки ошибок (`@steer/types/errors`)
- Структурированное логирование (`@steer/types/logger`)
- Кастомные React хуки:
  - `useDebounce` - для debounce значений
  - `useLocalStorage` - для работы с localStorage
  - `useAudioDevice` - для управления аудио устройством
  - `useAudioPipeline` - для управления аудио пайплайном
- Утилиты для оптимизации производительности (memoize, debounce, throttle)
- Тесты для store и middleware
- Скрипты для typecheck и lint во всех пакетах

### Изменено

- **OpenAI Service** - добавлена поддержка режимов и контекста
- **App Store** - добавлены действия для управления режимами и контекстом
- **API Routes** - обновлены для поддержки параметров режима
- **Типы** - добавлены `AssistantMode`, `InterviewContext`, `GenerateRequest`
- Улучшена типизация во всех пакетах
- Оптимизирована структура проекта
- Обновлены зависимости до последних версий
- README обновлен с информацией о режимах собеседования

### Исправлено

- Улучшена обработка ошибок в аудио пайплайне
- Оптимизирована работа с localStorage
- Исправлены проблемы с типизацией в store
