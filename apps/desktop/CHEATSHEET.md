# 📋 Шпаргалка: VB-Cable устройства

## 🎯 Главное правило

```
ДЛЯ ВЫВОДА ЗВУКА    →  CABLE Input
ДЛЯ ЗАХВАТА ЗВУКА   →  CABLE Output
```

## 📊 Таблица устройств

| Устройство | Где выбирать | Для чего |
|------------|--------------|----------|
| **CABLE Input** | Windows "Параметры звука" → Вывод | Воспроизведение звука |
| **CABLE Output** | Наше приложение → Компонент отладки | Захват звука |

## 🔄 Схема работы

```
┌─────────────────────────────────────────────────────────┐
│                    ВАШЕ ПРИЛОЖЕНИЕ                      │
│              (Zoom, Teams, Chrome, и т.д.)              │
└────────────────────┬────────────────────────────────────┘
                     │ воспроизводит звук
                     ↓
┌─────────────────────────────────────────────────────────┐
│              CABLE Input (VB-Audio)                     │
│         (Устройство воспроизведения/вывода)             │
│                                                          │
│  Выбирается в: Windows → Параметры звука → Вывод       │
└────────────────────┬────────────────────────────────────┘
                     │ передает звук внутри VB-Cable
                     ↓
┌─────────────────────────────────────────────────────────┐
│              CABLE Output (VB-Audio)                    │
│           (Устройство записи/захвата)                   │
│                                                          │
│  Выбирается в: Наше приложение → Компонент отладки     │
└────────────────────┬────────────────────────────────────┘
                     │ захватывает звук
                     ↓
┌─────────────────────────────────────────────────────────┐
│              НАШЕ ПРИЛОЖЕНИЕ                            │
│         (Voice Assistant Overlay)                       │
│                                                          │
│  Записывает → Транскрибирует → Отправляет в ChatGPT   │
└─────────────────────────────────────────────────────────┘
```

## ⚙️ Настройка за 3 шага

### Шаг 1: Windows (вывод звука)
```
Win + I → Система → Звук → Вывод
↓
Выбрать: CABLE Input (VB-Audio Virtual Cable)
```

### Шаг 2: Наше приложение (захват звука)
```
Кнопка "🔧 Отладка" → Устройства
↓
Выбрать: CABLE Output (VB-Audio Virtual Cable) (Output/Loopback)
```

### Шаг 3: Проверка
```
Воспроизвести звук → Проверить уровень аудио > 0%
```

## ❌ Типичные ошибки

### Ошибка 1: Выбрали CABLE Input в приложении
```
❌ НЕПРАВИЛЬНО:
   Приложение → CABLE Input
   
✅ ПРАВИЛЬНО:
   Приложение → CABLE Output
```

### Ошибка 2: Выбрали CABLE Output в Windows
```
❌ НЕПРАВИЛЬНО:
   Windows → Вывод → CABLE Output
   
✅ ПРАВИЛЬНО:
   Windows → Вывод → CABLE Input
```

### Ошибка 3: Выбрали (Input) вместо (Output/Loopback)
```
❌ НЕПРАВИЛЬНО:
   CABLE Output (VB-Audio) (Input)
   
✅ ПРАВИЛЬНО:
   CABLE Output (VB-Audio) (Output/Loopback)
```

## 💡 Мнемоника

Запомните так:

```
Windows ВВОДИТ звук В кабель    → CABLE Input
Приложение ВЫВОДИТ звук ИЗ кабеля → CABLE Output
```

Или:

```
IN-put  = звук идет ВНУТРЬ кабеля  (Windows)
OUT-put = звук идет НАРУЖУ кабеля  (Приложение)
```

## 🎯 Быстрая проверка

Если не работает, проверьте:

- [ ] Windows → Вывод → **CABLE Input** ✅
- [ ] Приложение → Устройства → **CABLE Output (Output/Loopback)** ✅
- [ ] Звук воспроизводится
- [ ] Уровень аудио > 0%
- [ ] Размер буфера растет
