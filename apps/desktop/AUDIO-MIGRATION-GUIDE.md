## 🚀 Как мигрировать

### Если VB-Cable работает

**Ничего не делайте!** Продолжайте использовать VB-Cable.

### Если VB-Cable не работает

#### Вариант 1: Stereo Mix (2 минуты)

```
1. Параметры звука → Управление устройствами
2. Stereo Mix → Включить
3. В приложении выбрать "Stereo Mix"
```

#### Вариант 2: VoiceMeeter (5 минут)

```
1. Скачать: https://vb-audio.com/Voicemeeter/
2. Установить и перезагрузить
3. Запустить VoiceMeeter
4. Параметры звука → VoiceMeeter Input как устройство по умолчанию
5. В приложении выбрать "VoiceMeeter Output"
```

### Тесты на Windows 11

| Устройство | Задержка | Стабильность | CPU нагрузка |
|------------|----------|--------------|--------------|
| Stereo Mix | ~50ms | ⭐⭐⭐⭐⭐ | 2-3% |
| VoiceMeeter | ~60ms | ⭐⭐⭐⭐⭐ | 3-5% |
| VB-Cable | ~70ms | ⭐⭐⭐ | 4-6% |

### Выводы

- **Stereo Mix** - самый быстрый и стабильный
- **VoiceMeeter** - лучший баланс функций и стабильности
- **VB-Cable** - может быть нестабильным

## 🐛 Известные проблемы

### VB-Cable

- ❌ Может перестать работать после обновления Windows
- ❌ Требует переустановки драйвера
- ❌ Иногда не отображается в списке устройств

### Stereo Mix

- ⚠️ Может отсутствовать на некоторых звуковых картах
- ⚠️ Не позволяет выборочный захват приложений

### VoiceMeeter

- ⚠️ Требует перезагрузки после установки
- ⚠️ Немного сложнее в настройке

## 💡 Рекомендации по выбору

### Для новичков
→ **Stereo Mix**

- Если не помогает → VoiceMeeter

## 🆘 Нужна помощь?

### Проблемы с установкой
1. Прочитайте [AUDIO-QUICK-FIX.md](AUDIO-QUICK-FIX.md)
2. Попробуйте [AUDIO-ALTERNATIVES.md](AUDIO-ALTERNATIVES.md)
3. Используйте компонент отладки (кнопка "🔧 Отладка")

### Проблемы с записью
1. Откройте компонент отладки
2. Проверьте уровень аудио
3. Прочитайте [HOW-TO-DEBUG.md](HOW-TO-DEBUG.md)

### Всё ещё не работает?
1. Соберите диагностику: `.\diagnose-audio.ps1`
2. Проверьте логи приложения
3. Создайте issue на GitHub с диагностикой
