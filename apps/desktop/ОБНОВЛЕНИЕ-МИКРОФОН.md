## Как использовать

### Для записи с микрофона:

1. Откройте компонент отладки в приложении
2. Выберите устройство: `CABLE Output (VB-Audio Virtual Cable) (Input)`
3. Настройте микрофон на вывод в CABLE Input
4. Нажмите "Начать захват"

### Для записи системного звука:

1. Откройте компонент отладки в приложении
2. Выберите устройство: `CABLE Output (VB-Audio Virtual Cable) (Output/Loopback)`
3. Установите CABLE Input как устройство воспроизведения
4. Нажмите "Начать захват"

## Технические детали

### Код уже поддерживал оба типа устройств

Rust код в `audio.rs` уже проверял как input, так и output устройства:

```rust
// Check input devices
if let Ok(devices) = host.input_devices() { ... }

// Check output devices (for loopback like VB-Cable)
if let Ok(devices) = host.output_devices() { ... }
```

### Что было изменено

Только **UI и документация**:

- Убрано сообщение об ошибке для (Input) устройств
- Добавлено пояснение, что (Input) используется для микрофона
- Создана подробная документация с примерами

### Что НЕ изменилось

- Rust backend (`audio.rs`, `commands.rs`) - без изменений
- Логика захвата аудио - без изменений
- API команды - без изменений
