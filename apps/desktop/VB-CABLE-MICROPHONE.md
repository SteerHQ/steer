# 🎤 Запись с микрофона через VB Cable

## Быстрый старт

Для записи звука **С МИКРОФОНА** через VB Cable:

### 1. Выбор правильного устройства

В компоненте отладки выберите:

```
CABLE Output (VB-Audio Virtual Cable) (Input)
                                       ^^^^^^
                                       ДЛЯ МИКРОФОНА!
```

**НЕ** выбирайте `(Output/Loopback)` - это для системного звука!

### 2. Настройка микрофона

#### Вариант А: Физический микрофон → VB Cable

1. Откройте **"Параметры звука"** (Win + I → Система → Звук)
2. В разделе **"Ввод"** выберите ваш физический микрофон
3. Откройте **"Дополнительные параметры звука"**
4. Найдите приложение, которое использует микрофон
5. Установите вывод на **"CABLE Input (VB-Audio Virtual Cable)"**

#### Вариант Б: Прямая запись с VB Cable Input

Если ваше приложение уже настроено на вывод в VB Cable:

1. В компоненте отладки выберите `CABLE Output (VB-Audio Virtual Cable) (Input)`
2. Нажмите "Начать захват"
3. Говорите в микрофон
4. Уровень аудио должен меняться

### 3. Проверка

После запуска захвата:

```
✅ Уровень аудио меняется при разговоре
✅ Размер буфера растет
✅ В логах: "Захват запущен успешно"
```

## Разница между устройствами

### 🎤 CABLE Output (Input) - ДЛЯ МИКРОФОНА

```
Что захватывает: Звук, который ЗАПИСЫВАЕТСЯ в VB Cable
Использование: Запись голоса, микрофона
Пример: Ваш голос в Zoom/Discord → VB Cable → Приложение
```

### 🔊 CABLE Output (Output/Loopback) - ДЛЯ СИСТЕМНОГО ЗВУКА

```
Что захватывает: Звук, который ВОСПРОИЗВОДИТСЯ из VB Cable
Использование: Запись музыки, видео, системных звуков
Пример: YouTube → VB Cable → Приложение
```

## Типичные сценарии

### Сценарий 1: Запись своего голоса

**Цель:** Записать то, что вы говорите в микрофон

**Настройка:**

1. Физический микрофон → Приложение (Zoom, Discord)
2. Приложение → CABLE Input (VB-Audio Virtual Cable)
3. Наше приложение → CABLE Output (Input) ✅

**Результат:** Записывается ваш голос

### Сценарий 2: Запись разговора (оба участника)

**Цель:** Записать и ваш голос, и собеседника

**Настройка:**

1. Установите VB Cable как устройство по умолчанию для вывода
2. В Zoom/Discord установите:
   - Микрофон: Ваш физический микрофон
   - Динамики: CABLE Input (VB-Audio Virtual Cable)
3. В нашем приложении выберите: CABLE Output (Output/Loopback) ✅

**Результат:** Записывается весь разговор

### Сценарий 3: Запись только собеседника

**Цель:** Записать только то, что говорит собеседник

**Настройка:**

1. В Zoom/Discord установите:
   - Динамики: CABLE Input (VB-Audio Virtual Cable)
2. В нашем приложении выберите: CABLE Output (Output/Loopback) ✅

**Результат:** Записывается только собеседник

## Отладка

### Проблема: Нет звука при записи с микрофона

**Проверка 1:** Правильное устройство?

```
✅ CABLE Output (VB-Audio Virtual Cable) (Input)
❌ CABLE Output (VB-Audio Virtual Cable) (Output/Loopback)
```

**Проверка 2:** Звук идет в VB Cable?

1. Откройте микшер громкости Windows
2. Найдите приложение (Zoom, Discord)
3. Убедитесь, что вывод установлен на "CABLE Input"

**Проверка 3:** Микрофон работает?

1. Откройте "Параметры звука" → "Проверка микрофона"
2. Говорите - полоса должна двигаться

### Проблема: Записывается системный звук вместо микрофона

**Причина:** Выбрано устройство (Output/Loopback)

**Решение:** Выберите устройство (Input)

```powershell
Get-PnpDevice -Class "AudioEndpoint" | Where-Object {$_.FriendlyName -like "*Cable*"}
```

## Краткая справка

| Что записать   | Устройство        | Настройка                               |
| -------------- | ----------------- | --------------------------------------- |
| Свой голос     | (Input)           | Микрофон → Приложение → VB Cable        |
| Собеседника    | (Output/Loopback) | Приложение → VB Cable → Наше приложение |
| Оба            | (Output/Loopback) | Все через VB Cable                      |
| Системный звук | (Output/Loopback) | Звук → VB Cable                         |
