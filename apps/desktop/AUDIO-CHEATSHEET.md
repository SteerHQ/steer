# 📋 Шпаргалка: Настройка аудио

## 🎯 Цель
Захватить системный звук для распознавания речи на собеседовании.

---

## ✅ Вариант 1: Stereo Mix (САМЫЙ ПРОСТОЙ)

### Когда использовать
- У вас Windows 10/11
- Не хотите ничего устанавливать
- Нужно быстрое решение

### Как включить
```
1. Правый клик на иконке звука → "Параметры звука"
2. "Управление звуковыми устройствами"
3. Найти "Stereo Mix" → "Включить"
4. В приложении выбрать "Stereo Mix"
```

### Плюсы
✅ Встроен в Windows  
✅ Не требует установки  
✅ Захватывает ВСЕ звуки автоматически  

### Минусы
⚠️ Может отсутствовать на некоторых звуковых картах

---

## ✅ Вариант 2: VoiceMeeter (РЕКОМЕНДУЕТСЯ)

### Когда использовать
- Нет Stereo Mix
- Нужны гибкие настройки
- Хотите стабильное решение

### Как установить
```
1. Скачать: https://vb-audio.com/Voicemeeter/
2. Установить → ПЕРЕЗАГРУЗИТЬ компьютер
3. Запустить VoiceMeeter один раз
4. Параметры звука → "VoiceMeeter Input" как устройство по умолчанию
5. В приложении выбрать "VoiceMeeter Output"
```

### Плюсы
✅ Бесплатный  
✅ Очень стабильный  
✅ Гибкие настройки  
✅ Работает на всех системах  

### Минусы
⚠️ Требует установки и перезагрузки

---

## ⚠️ Вариант 3: VB-Cable (НЕ РЕКОМЕНДУЕТСЯ)

### Когда использовать
- Уже установлен и работает
- Нет других вариантов

### Проблемы
❌ Может быть нестабильным  
❌ Проблемы с драйверами  
❌ Требует частой переустановки  

### Если не работает
→ Переключитесь на VoiceMeeter или Stereo Mix

---

## 🔍 Проверка работы

### В компоненте отладки
```
1. Открыть "🔧 Отладка"
2. Выбрать устройство
3. "▶️ Начать захват"
4. Включить музыку
5. Проверить: Уровень аудио > 10%
```

### Если уровень 0%
- **Stereo Mix:** Убедитесь, что включен
- **VoiceMeeter:** Проверьте устройство по умолчанию
- **VB-Cable:** Попробуйте другое решение
