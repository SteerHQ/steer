# ⚡ Быстрое решение проблем с VB-Cable

## 🚨 VB-Cable не работает?

### Решение за 2 минуты: Используйте Stereo Mix

1. **Откройте настройки звука:**
   - Правый клик на иконке звука в трее
   - "Параметры звука"

2. **Включите Stereo Mix:**
   - "Управление звуковыми устройствами"
   - Найдите "Stereo Mix"
   - Нажмите "Включить"

3. **В приложении:**
   - Откройте "🔧 Отладка"
   - Нажмите "🔄 Обновить список"
   - Выберите "Stereo Mix"
   - Нажмите "▶️ Начать захват"

4. **Проверьте:**
   - Включите музыку
   - Уровень аудио должен быть > 10%

**Готово!** Stereo Mix захватывает все звуки системы автоматически.

---

## 🔧 Stereo Mix отсутствует?

### Установите VoiceMeeter (5 минут)

1. **Скачайте:**
   - https://vb-audio.com/Voicemeeter/
   - Выберите "VoiceMeeter" (базовая версия)

2. **Установите:**
   - Запустите установщик
   - **ОБЯЗАТЕЛЬНО перезагрузите компьютер**

3. **Настройте:**
   - Запустите VoiceMeeter один раз
   - Откройте "Параметры звука" Windows
   - Установите "VoiceMeeter Input" как устройство по умолчанию

4. **В приложении:**
   - Выберите "VoiceMeeter Output"
   - Начните захват
