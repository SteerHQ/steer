# WASAPI Loopback - Быстрый старт

## Что это?

WASAPI Loopback - это встроенная функция Windows для захвата системного аудио **БЕЗ установки виртуальных устройств**.

## Как использовать (3 шага)

### 1. Запустите приложение

```bash
bun run dev
# или
bun run tauri:dev
```

### 2. Выберите WASAPI Loopback

В списке аудиоустройств выберите:
```
WASAPI Loopback (System Audio - No Virtual Device Needed)
```

### 3. Начните захват

Нажмите "Начать захват" - готово! Приложение автоматически захватывает все системное аудио.

## Для Zoom/Google Meet

1. Запустите Zoom или Google Meet
2. Начните звонок
3. Голос собеседника автоматически захватывается
4. Используйте распознавание речи как обычно

## Преимущества

✅ Не требует установки VB-Cable, VoiceMeeter или Stereo Mix  
✅ Работает "из коробки" на Windows 10/11  
✅ Захватывает все системное аудио автоматически  
✅ Более стабильно чем виртуальные устройства  

## Если не работает

1. Убедитесь, что используете Windows 10/11
2. Проверьте, что звук воспроизводится в системе
3. Перезапустите приложение
