# 🎤 Быстрая настройка микрофона VB Cable

## Что нужно сделать

### 1. Выбрать правильное устройство

В приложении откройте компонент отладки и выберите:

```
CABLE Output (VB-Audio Virtual Cable) (Input)  ← ДЛЯ МИКРОФОНА
```

**НЕ выбирайте** `(Output/Loopback)` - это для системного звука!

### 2. Настроить микрофон

#### Если хотите записать свой голос:

1. Откройте **Параметры звука** (Win + I)
2. В разделе **"Ввод"** выберите ваш микрофон
3. В приложении (Zoom, Discord и т.д.) установите вывод на **CABLE Input**

#### Если хотите записать разговор (оба участника):

1. В Zoom/Discord установите:
   - **Микрофон**: Ваш физический микрофон
   - **Динамики**: CABLE Input (VB-Audio Virtual Cable)
2. В нашем приложении выберите: **CABLE Output (Output/Loopback)**

### 3. Проверить

После запуска:

- ✅ Уровень аудио меняется
- ✅ Размер буфера растет
- ✅ В логах: "Захват запущен успешно"

## Разница между устройствами

| Устройство            | Что записывает                                | Когда использовать                  |
| --------------------- | --------------------------------------------- | ----------------------------------- |
| **(Input)**           | Звук, который **записывается** в VB Cable     | Запись с микрофона                  |
| **(Output/Loopback)** | Звук, который **воспроизводится** из VB Cable | Запись системного звука, разговоров |

## Проблемы?

### Нет звука при записи

1. Проверьте, что выбрано правильное устройство
2. Убедитесь, что звук идет в VB Cable (проверьте микшер Windows)
3. Проверьте, что микрофон работает
