## 📖 Основные концепции

### Режимы работы

Voice Assistant поддерживает 4 режима:

| Режим | Иконка | Назначение | Особенности |
|-------|--------|------------|-------------|
| Собеседование | 🎯 | Технические интервью | Краткие ответы с контекстом |
| Алгоритмы | 🧮 | Coding challenges | Сложность O(n), псевдокод |
| Шпаргалка | 📝 | Быстрые справки | Только факты |
| Общий | 💬 | Стандартные вопросы | Баланс краткости и полноты |

### Контекстная память

В режиме собеседования ассистент:
- Запоминает последние 10 вопросов
- Использует контекст для релевантных ответов
- Автоматически очищает историю при переключении

## 🎯 Типичные сценарии

### Frontend Interview
```
1. Запустите в режиме 🎯 Собеседование
2. Теоретические вопросы → оставайтесь в режиме
3. Coding challenge → переключите на 🧮 Алгоритмы
4. Синтаксис → переключите на 📝 Шпаргалка
```

### Backend Interview
```
1. Запустите в режиме 🎯 Собеседование
2. Вопросы по БД/API → оставайтесь в режиме
3. Алгоритмы → переключите на 🧮 Алгоритмы
4. SQL команды → переключите на 📝 Шпаргалка
```

### Алгоритмическое интервью
```
1. Запустите в режиме 🧮 Алгоритмы
2. Получайте подсказки по сложности
3. Смотрите псевдокод
4. Для определений → переключите на 📝 Шпаргалка
```

## 💡 Советы

### ✅ Делайте

- Формулируйте вопросы четко и конкретно
- Используйте правильный режим для типа вопроса
- Очищайте историю при смене темы
- Держите окно поверх других окон

### ❌ Избегайте

- Слишком широких вопросов ("Расскажи про React")
- Многосоставных вопросов
- Неконкретных формулировок ("Помоги с задачей")
