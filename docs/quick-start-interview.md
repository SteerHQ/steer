# Быстрый старт: Использование на собеседовании

## За 5 минут до собеседования

### 1. Запуск и настройка (2 минуты)

```bash
# Запустите приложение
VoiceAssistant.exe

# При первом запуске:
1. Введите OpenAI API ключ
2. Выберите CABLE Output как аудиоустройство
3. Выберите режим "Собеседование" по умолчанию
4. Сохраните настройки
```

### 2. Проверка работы (1 минута)

```
✅ Визуализатор показывает активность
✅ Статус: "Capturing"
✅ Режим: 🎯 Собеседование
```

### 3. Тестовый вопрос (1 минута)

```
Скажите в микрофон: "Что такое замыкание в JavaScript?"
→ Проверьте ответ в overlay окне
→ Убедитесь что ответ краткий (2-3 предложения)
```

### 4. Готово! (1 минута)

```
✅ Приложение работает
✅ Режим собеседования активен
✅ Можно начинать интервью
```

## Во время собеседования

### Быстрые действия

| Действие | Как сделать |
|----------|-------------|
| Переключить режим | Клик на 🎯 → выбрать режим |
| Очистить историю | Клик на 🎯 → "Очистить историю" |
| Очистить чат | Кнопка "🗑️ Очистить чат" |
| Открыть настройки | Кнопка "⚙️ Настройки" |

### Когда переключать режимы

```
Теоретические вопросы → 🎯 Собеседование
Coding challenge       → 🧮 Алгоритмы
Быстрые факты         → 📝 Шпаргалка
Общие вопросы         → 💬 Общий
```

---

## Типичные сценарии

### Сценарий 1: Frontend Interview

```
1. Запустите в режиме 🎯 Собеседование
2. Отвечайте на теоретические вопросы
3. При coding challenge → переключите на 🧮 Алгоритмы
4. Для синтаксиса → переключите на 📝 Шпаргалка
```

### Сценарий 2: Backend Interview

```
1. Запустите в режиме 🎯 Собеседование
2. Для вопросов по БД/API → оставайтесь в режиме собеседования
3. Для алгоритмов → переключите на 🧮 Алгоритмы
4. Для SQL/команд → переключите на 📝 Шпаргалка
```

### Сценарий 3: Алгоритмическое интервью

```
1. Запустите в режиме 🧮 Алгоритмы
2. Получайте подсказки по сложности O(n)
3. Смотрите псевдокод в ответах
4. Для определений → переключите на 📝 Шпаргалка
```

## Советы для эффективного использования

### ✅ Делайте

- Формулируйте вопросы четко и конкретно
- Используйте правильный режим для типа вопроса
- Очищайте историю при смене темы
- Держите окно поверх других окон

### ❌ Избегайте

- Слишком широких вопросов ("Расскажи про React")
- Многосоставных вопросов
- Неконкретных формулировок ("Помоги с задачей")

## Горячие клавиши (планируется)

```
Ctrl+1 → Режим собеседования
Ctrl+2 → Режим алгоритмов
Ctrl+3 → Режим шпаргалки
Ctrl+4 → Общий режим
Ctrl+H → Очистить историю
Ctrl+C → Очистить чат
```

## Troubleshooting

### Проблема: Не слышу звук

**Решение:**
1. Проверьте настройку прослушивания VB-Cable
2. Убедитесь что выбраны правильные динамики

### Проблема: Ответы слишком длинные

**Решение:**
1. Переключитесь на режим шпаргалки
2. Переформулируйте вопрос конкретнее

### Проблема: Нет контекста в ответах

**Решение:**
1. Убедитесь что используете режим собеседования (🎯)
2. Проверьте что история не пустая (клик на режим)
