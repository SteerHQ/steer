# Что изменилось: Режимы для собеседований

## 🎯 Главное

Voice Assistant теперь **специально оптимизирован для технических собеседований** с 4 режимами работы и контекстной памятью.

## ⚡ Быстрый обзор

### Что добавлено

1. **4 режима работы** (переключение одним кликом):
   - 🎯 Собеседование - краткие ответы с контекстом
   - 🧮 Алгоритмы - помощь с задачами + сложность O(n)
   - 📝 Шпаргалка - только факты
   - 💬 Общий - стандартный режим

2. **Контекстная память** (режим собеседования):
   - Запоминает последние 10 вопросов
   - Более умные ответы с учетом контекста
   - Автоочистка при переключении режима

3. **Новый UI компонент**:
   - Выпадающее меню для выбора режима
   - Счетчик вопросов в истории
   - Кнопка очистки истории

## 🚀 Как использовать

### Для пользователей

```
1. Запустите приложение
2. Кликните на режим (по умолчанию 🎯)
3. Выберите нужный режим
4. Задавайте вопросы
```

### Для разработчиков

```typescript
// Переключение режима
setMode('interview');

// Добавление в контекст
addToInterviewContext(question, answer);

// Генерация ответа
const response = await interviewService.generateResponse({
  transcript: question,
  mode: 'interview',
  context: getInterviewContext(),
  apiKey: apiKey,
});
```
